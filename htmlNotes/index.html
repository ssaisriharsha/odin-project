HTML basics

Doctype declaration

HTML stands for hyper text markup language. Every html doc starts with a doctype declaration. The doctype declaration tells the browser which version of html to use. Latest version is html5 and to use it, we simply type DOCTYPE html as shown below.

<!DOCTYPE html>

Html, head and body tags

The html tag is used to tell the browser that the following document is html document. html tag can have lang attribute which specifies the language used to display the webpage. Although lang is an optional attribute, using it allows screen readers to recognize the language and give correct pronounciation.

The head tag contains the information about the page. It contains the title of the webpage, charset used etc.

The body element contain the information to be displayed in the webpage.

Paragraphs, headings, links and images

To write a paragraph we use <p>paragraph</p>. To go to next line, we use <br> tag. the linebreak tag is a self closing tag. It doesn't require a closing tag.

Coming to headings, there are six types of headings ranging from <h1> to <h6>. The size decreases as we go from h1 to h6.

To embed a link, we use anchor element <a> it takes href as attribute which stands for hypertext reference. The link to be opened is contained in href attribute. If we want to open the webpage in newtab then we set the target attribute to "_blank" and then set the rel attribute to "noopener noreferrer".

To embed images, we use <img> tag. It is a self closing tag. The location of the image is specified using src attribute. The location can be local or remote. To provide alternate text we use alt attribute.

Ordered and Unordered lists

To create a ordered list, we use ol opening and closing tags. The list elements are created using li opening and closing tag. To create Unordered list, we use ul opening and closing tags.
